# Portafolio

Portafolio one-page con proyectos reales, demos y enlaces al codigo. Enfocado en
mostrar el trabajo de forma clara y directa.

## Demo

- Produccion: https://portafolio-mu-liart.vercel.app/

## Instalacion

Requisitos: Node.js 18+

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Este proyecto se despliega en Vercel conectado al repo de GitHub.
Con cada push a `main`, Vercel construye y publica automaticamente.
